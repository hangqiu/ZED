##### 
#####   This sample demonstrates how to grab images and disparity map with the ZED SDK
#####   and apply the result in a 3D view "point cloud style" with OpenGL /freeGLUT.                   
##### 
#####   This sample needs freeGLUT (OpenGL Window).
#####
#####   =====================
#####   * Build the program *
#####   =====================
#####    
#####       Open a terminal in openGL 3D Viewer directory and execute the following command:
#####           $ mkdir build
#####           $ cd build
#####           $ cmake ..
#####           $ make
#####    
#####   ===================
#####   * Run the program *
#####   ===================
#####    
#####       Open a terminal in build directory and execute the following command:
#####           $ ./ZED\ openGL\ 3D\ Viewer
